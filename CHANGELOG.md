# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-06

### Added

- Modular Python architecture with separated concerns
- Comprehensive type hints throughout codebase
- Robust error handling with specific exception types
- Configuration module for centralized settings
- Automatic PDF compilation with visual loading bar
- Interactive checkboxes (checkmark ✓ and cross ✗)
- Mini calendars with highlighted event days
- Course codes in event listings
- Professional README with usage examples
- MIT License
- Input validation for JSON data structure
- Proper Python package structure with `__init__.py`

### Features

- JSON to LaTeX conversion
- Automatic date parsing and validation
- Monthly event grouping
- Exam period section support
- TBA (To Be Announced) event handling
- Custom room assignments per event
- Default room assignments per subject
- Landscape-oriented professional formatting
- Automatic cleanup of auxiliary LaTeX files

### Technical

- Python 3.x compatible
- No external dependencies (standard library only)
- Zero linting errors
- Comprehensive docstrings
- Type-annotated functions
- Modular code organization
